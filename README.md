# Sample Telegram Trivia Bot

### Installation
```bash
git clone https://github.com/tanseersaji/telegramQuizBot.git
cd telegramQuizBot
npm install
```
### Telegram Bot Token
Refer to this blog, on how to get Telegram Bot Token

```bash
export BOT_TOKEN=<YOUR TELEGRAM API TOKEN>
```
### Start the server
```bash
node main.js
```
